Questions fréquentes

Où sont stockées mes données ?

Uniquement sur votre iPhone, via SwiftData. LifeOS n'a ni compte, ni serveur, ni cloud : personne d'autre que vous n'y a accès. Détails dans la politique de confidentialité.

Comment supprimer toutes mes données ?

Désinstallez simplement l'application : iOS efface alors la totalité des données locales de LifeOS. Il n'existe aucune copie ailleurs.

Les pas, le sommeil ou la HRV ne s'affichent pas.

Vérifiez que l'accès Santé est accordé : Réglages → Santé → Accès aux données et appareils → LifeOS. Sans autorisation, les modules concernés basculent en saisie manuelle. Notez qu'une Apple Watch ou un iPhone porté sur soi est nécessaire pour alimenter ces mesures.

Les notifications de rappel n'arrivent pas.

Autorisez les notifications dans Réglages → Notifications → LifeOS, et vérifiez que le mode Concentration n'est pas en train de les filtrer. Tous les rappels sont des notifications locales : aucune connexion n'est requise.

L'app fonctionne-t-elle sans connexion Internet ?

Oui, entièrement. LifeOS est conçue pour fonctionner 100 % hors ligne. Seules quelques fonctions optionnelles (météo pour les tenues, recherche d'aliments OpenFoodFacts) utilisent Internet lorsqu'il est disponible.

Mes données sont-elles incluses dans la sauvegarde iCloud ?

Si la sauvegarde iCloud de votre iPhone est activée, les données de LifeOS y sont incluses comme celles de toute app, chiffrées par Apple. Restaurer votre iPhone depuis cette sauvegarde restaure aussi vos données LifeOS.

Sur quels appareils l'app est-elle disponible ?

LifeOS nécessite un iPhone sous iOS 17 ou ultérieur.

Comment signaler un bug efficacement ?

Indiquez votre modèle d'iPhone, votre version d'iOS, le module concerné et les étapes pour reproduire le problème. Une capture d'écran aide énormément. Envoyez le tout par e-mail ou via une issue GitHub.
